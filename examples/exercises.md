## Exercicio 01 - Primeiros passos com Ruby

**Objetivo:** Criar um pequeno script para praticar variaveis, entrada e saida.

**Instrucao:**
Crie um script Ruby que:

1. leia o nome de uma pessoa
2. leia a idade
3. imprima uma frase combinando as duas informacoes

**Criterios de sucesso:**

- [ ] Usa variaveis para armazenar os dados
- [ ] Lê entrada do terminal
- [ ] Imprime uma mensagem clara para a pessoa usuaria
