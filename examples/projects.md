# Projetos

Projetos praticos para consolidar o aprendizado de Ruby.

## Projeto 01 - Cadastro simples no terminal

**Objetivo:** Criar um pequeno programa de terminal para cadastrar pessoas em memoria.

**Requisitos:**

- [ ] Permite cadastrar nome e idade
- [ ] Lista os registros cadastrados
- [ ] Organiza a logica em metodos
- [ ] Roda no terminal sem dependencias externas
