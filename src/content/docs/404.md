Algumas razões pelas quais isso pode ter acontecido:

- O método foi renomeado e a URL antiga não tem um `alias` configurado
- Você seguiu um link externo que aponta para uma versão anterior do site
- O objeto foi coletado pelo garbage collector - a página foi removida
